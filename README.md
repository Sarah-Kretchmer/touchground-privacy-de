  <p>Die App verwendet TelemetryDeck, einen datenschutzorientierten Analysedienst, um anonyme Nutzungsereignisse zu erfassen. Dies hilft mir zu verstehen, welche Funktionen genutzt werden und die Funktionalität im Laufe der Zeit zu verbessern.</p>

    <li>Gesundheitsdaten</li>

  </ul>
  <p>Telemetriesignale können allgemeine, nicht-personenbezogene Details enthalten wie:</p>
  <ul>

    <li>welcher Bildschirm geöffnet wurde</li>

  </ul>
  <p>TelemetryDeck erhält <strong>nicht</strong>:</p>
